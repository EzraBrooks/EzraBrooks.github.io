Welcome to the Red Clay Game Jam page!

The Red Clay Game Jam will be an annual game jam dedicated to producing small games that represent the Red Clay district's annual motto. For example, the 2013-2014 school year's motto is "Let's Go Higher," therefore that will be the topic for the game jam. Stay tuned for details on the rules and schedule for this year's jam!
